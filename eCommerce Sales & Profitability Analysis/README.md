# Superstore Sales Analysis

This project performs a comprehensive analysis of an eCommerce "Superstore" dataset to uncover sales trends, identify the most profitable products and regions, and understand how business factors like discounts impact the bottom line.

The analysis was conducted using Python in a Jupyter Notebook environment, utilizing libraries such as:

- Pandas for data manipulation
- Matplotlib for data visualization
- Seaborn for advanced visualizations

## Dataset

The dataset used in this analysis is the "Sample Superstore" dataset, originally sourced from Tableau.
